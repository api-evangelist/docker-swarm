# Docker Swarm (docker-swarm)
Native clustering and orchestration solution for Docker containers that turns a pool of Docker hosts into a single virtual host. Docker Swarm enables deploying applications across multiple machines and provides high availability and load balancing for containerized workloads.

## APIs

Docker Swarm does not publish a standalone OpenAPI specification. Swarm management endpoints (`/swarm`, `/services`, `/tasks`, `/nodes`, `/secrets`, `/configs`) are part of the Docker Engine API and are tracked in the [`docker`](../docker) repository (`openapi/docker-openapi.yml`, Engine API v1.54).
